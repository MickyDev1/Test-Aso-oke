# Aso-oke Website

Aso-oke is a modern storefront and storytelling site celebrating Yoruba woven textiles. It highlights curated collections, cultural context, and custom-order flows for handcrafted garments and accessories.

## Highlights
- Curated collections of Aso-oke fabrics and ready-to-wear pieces
- Editorial content on weaving techniques and heritage
- Custom order requests with measurements
- Sustainable sourcing and artisan partnerships
- International shipping and contact information

## Tech Stack
- Next.js (App Router) + React
- Tailwind CSS
- Firebase

## Getting Started
Install dependencies and run the development server:

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000` to view the site.

## Scripts
- `pnpm dev` - Run the local dev server
- `pnpm build` - Build for production
- `pnpm start` - Start the production server
- `pnpm lint` - Lint the codebase

## Project Structure
- `app/` - Routes and UI
- `components/` - Reusable UI components
- `lib/` - Helpers and utilities
- `public/` - Static assets

## Notes
Configuration for Firebase and other services should be provided via environment variables.
